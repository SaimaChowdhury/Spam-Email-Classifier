## How to Run This Notebook ##
------------------------------------------------------------

This notebook contains code for building an email spam classifier. To run the code:

1.  Ensure you have the `spam.csv` dataset available. If not, download it from the provided Kaggle link below-
		(https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset/suggestions).
2.  Run each code cell sequentially from top to bottom.
3.  The notebook will perform data loading, preprocessing, feature engineering, model training (Naive Bayes, Logistic Regression, Random Forest), and evaluation.
4.  The output of each cell will be displayed below it.
